== Installation ==

RO:	Pasii de urmat pentru instalarea pluginului:

    1. Descarcati pluginul, dezarhivati si urcati directorul tvwidget in folderul /wp-content/plugins/.
    2. Intrati in partea de administrare a blogului, Plugins si activati pluginul "TvWidget"
    3. In meniul principal veti gasi butonul "Tv Widget" de unde puteti administra categoriile care doriti sa fie afisate pe blogul vostru.
    4. Daca doriti sa afisati posturile din anumite categorii bifati categoriile respective si apoi apasati butonul Listeaza Posturi Tv. Din lista afisata bifati posturile care doriti sa le afisati in sidebarul blogului si apoi bifati "Salveaza Posturi Tv".
    5. Acum navigati la Appearance -> Widgets si alegeti pozitia widgetului in sidebar.

EN: Steps to follow to install plugin:

     1. Download the plugin, unzip and upload tvwidget directory in the folder / wp-content/plugins /.
     2. Enter the blog management, Plugins and activate the plugin "TvWidget"
     3. In the main menu you will find the "TV Widgets" where you can manage the categories to be displayed on your blog.
     4. If you want to display posts from certain categories check these categories and then press the List TV stations. Check posts from the list that you want to display in the sidebar blog and then check "Save TV".
     5. Now navigate to Appearance -> Widgets and widget in the sidebar choose position.
